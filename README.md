# Toyo_ExBprobe_Transmittancy
Toyo's codes for transmittancy matrix for ExB probe are here.

## What you can do
You can calculate a transmittancy matrix for your ExB probe.  

## What you need
1. All codes are written in Python (Version 3.13). You will have to set a Python environment with various packages.
2. Have your ExB probe's geometric data (ideally a CAD file, at least a list of geometric parameters).
3. Have your ExB probe's E- and B-field data.
4. Pick an ion you want to calculate.
5. Consider the velocity range for your ExB probe operation.
6. Pick your transmittancy matrix calculation methods.
7. Run the code, and you'll get a transmittancy matrix!
